# OpenSpec Dashboard

Interactive dashboard tracking AI-powered operator development — onboarding status, agentic pipeline metrics, and per-ticket telemetry across cert-manager, ZTWIM, SSCSI, Must-Gather, and ESO operators.

**Live Dashboard:** [https://anandkuma77.github.io/open-spec-dashboard/](https://anandkuma77.github.io/open-spec-dashboard/)

## Local Development

```bash
python3 -m http.server 8080
```

Open **http://localhost:8080** in your browser.
